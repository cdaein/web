Modular Design System One (MDS1) is a web app built with JavaScript technologies (HTML5 Canvas, p5.js, react.js and material-ui) that enables me to quickly create design compositions. 

I always disliked the traditional way of creating digital animation, where you define keyframes first, wait until render is done, and go back to more changes. While this approach has its own merit, What I wanted was the animation process to be more direct and spontaneous without much overhead. With my software, I can make changes to the composition and typography while it is being played. There is also chance encounters with different color palettes. I have also created a grid-based font specifically designed for this project. It has variable font-like characteristics. Everything is controlled programmatically. 

It is a work-in-progress and only for my personal use for now. Check out [examples](/tags/MDS1) of motion-typopgraphy compositions created with this software.
